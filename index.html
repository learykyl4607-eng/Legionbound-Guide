Core Mechanic

How Class Upgrades Work

Legionbound uses a proximity-based upgrade system. Classes are arranged in three synergy loops — place a class next to the one that follows it in the loop and it will teach that unit how to upgrade.

The Rule

Each class teaches the next class in its loop how to upgrade. Position the teacher adjacent to the student on the battlefield to trigger the upgrade. The arrows in the infographic show the teaching direction.

Third Unlock Method

Skill Tree — Optimal Unlock Order

Fourteen classes must be unlocked through the skill tree. The order you unlock them significantly affects how quickly you can build upgrade chains. Prioritize classes that sit at the head of the longest teaching sequences, and avoid spending points on classes that are covered by achievements.
